#### 補足
「式を『評価』する」という言い回しが出てくるが，これは数学においてよく用いられる言い回しである．

「評価」という単語をWikipediaで見てみると，意味の一つに
```
数学や計算機科学において、変数に関連づけられた値などをもとに関数などの式・表現が表す値を計算すること。あるいは、不等式により値の範囲を絞り込むこと。
```
とある．
つまりそういうことである（丸投げw）．

使い方の例としては
- 「1 + 2」という式を評価すると3になる．
- 「x = 3」のとき，「x + 20」を評価すると23である．
