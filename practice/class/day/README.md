## 補足

### 閏年
念のため閏年の定義を確認しておく．

西暦y年が閏年になるのは
- yが4で割り切れるとき
- しかし，yが100で割り切れるときは閏年ではなくなる
- しかし，yが400で割り切れるときは通常通り閏年となる

### 年月日から曜日を求める方法
年をy，月をm，日にちをdとする．

- mが1または2なら
	- yから1を引く
	- mに13を加える
- y + y / 4 - y / 100 + y / 400 + (13 * m + 8) / 5 + d を計算する（この値をaとおく）
- aを7で割った余りを求める（この余りをbとおく）
- b = 0なら日曜，1なら月曜， ... ，6なら土曜
